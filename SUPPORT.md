# Support

## Where To Ask For Help

- Use GitHub Issues for reproducible bugs, regressions, and build failures.
- Use GitHub Discussions if the repository enables it for usage questions or general guidance.
- Use the security path in `SECURITY.md` for security-sensitive reports.

## What To Include

Please include:

- NiceCLI version
- Windows version
- whether the problem is in startup, auth, quota, config, tray, or build flow
- relevant logs or screenshots
- exact verification or reproduction steps

## Before Opening A Report

- Re-run the maintained verification steps in `docs/maintenance.md` if you are changing code.
- Check whether the issue is local-only behavior, tray/host lifecycle, config persistence, or provider/runtime behavior.
- Confirm whether the problem still reproduces on the current repository baseline.
